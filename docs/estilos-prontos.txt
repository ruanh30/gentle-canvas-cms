================================================================================
                    ESTILOS PRONTOS — O QUE CADA UM ALTERA
================================================================================

Cada estilo pronto do Admin Menu aplica automaticamente um conjunto completo de
configurações do cabeçalho. Os presets foram projetados para serem
estruturalmente diferentes entre si, utilizando combinações distintas de layout,
tipografia, container, estados, busca, menu mobile e barra de anúncio.
